# 07_filter_map standalone app

## Run

```bash
cd 00_Proposal/07_filter_map
python -m pip install -r requirements.txt
python app.py
```

Then open `http://127.0.0.1:5050`.

## Notes

- UI uses Comfortaa from Google Fonts.
- Monochrome control styling with accent `#FBD124`.
- Bivariate fill colors remain data-driven.
